About the counter

A small window, refreshed constantly.

Every number on this page is borrowed from livecounts.io — a service that polls YouTube's public API at roughly two-second intervals and serves the result as a small embeddable widget.

The widget you see in the hero is the actual counter, not a screenshot and not a recreation. It updates in place, in your browser, every time YouTube reports a new subscriber event for the channel.

The channel ID UC7c8wbA9yQjzxkj2uyrHrkg is the unique identifier YouTube uses for Steak. Paste it into any YouTube URL and you'll land on the channel itself; paste it into a counter service and you get exactly what's rendered above.

Subscribers aren't the only measure of a channel, but they are the most public one — the number quoted in interviews, posted on thumbnails, celebrated in milestone videos. This page is a quieter version of that celebration.

Notes

A handful of things worth knowing.

If the counter appears blank or shows an error page, the most common cause is the page being opened directly from your computer via the file:// protocol — many embed services refuse to load inside a file with no real referrer. Host the file on any web server (GitHub Pages, Netlify, or a local python -m http.server) and the iframe will load normally.

The second most common cause is a browser extension — ad-blockers and privacy shields (uBlock Origin, Privacy Badger, Brave Shields) frequently treat third-party iframes from livecounts.io as tracking and silently drop them. Disabling the extension for this page usually resolves it.

YouTube publicly rounds subscriber counts above 1,000 — most third-party counters display the same rounded figure, which can occasionally lag the precise number shown on the channel page itself.

If livecounts.io is having an outage, the backup link under the counter points to socialcounts.io, which serves the same data from a different provider.
